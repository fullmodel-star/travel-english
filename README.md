# ✈️ 旅遊英文（全家出國會話）

一個**離線可用、免安裝、零追蹤**的旅遊生活會話 PWA，給全家出國時「現場照著念就能用」的英文短句，附中文諧音、跨文化備註與救命短句。

> 純單一 HTML 檔、資料全存本機、不上傳雲端、免註冊、無廣告。

> **狀態**：✅ 已上線　｜　v6　｜　🔷 靛藍主題（theme `#4f46e5`）　｜　**23 情境 / 426 句（諧音 100%）**　｜　📱 響應式
> 上線網址：<https://fullmodel-star.github.io/travel-english/>　｜　已收錄於[英語學習入口 hub](../14_英語學習入口/)

---

## ✨ 功能
- **23 情境、426 句**：每句含英文、中文、**中文諧音（100% 覆蓋）**、朗讀
- **跨文化備註**：各國禮儀／禁忌提醒
- **分眾**：依家庭成員（大人／長輩／小孩）過濾情境
- **救命短句**：緊急、就醫、走失等一鍵直達
- **口說誠實回饋**：錄音比對、老實給分不灌水
- **兒童遊戲層**：小朋友互動關卡
- **行前精靈**：出發前的準備引導
- **個人化資料卡**：護照／保險／聯絡人（單檔離線保存）

## 🚀 使用
瀏覽器開 `index.html` 即可；安裝成 App 需用上線 https 網址（Android Chrome →「安裝應用程式」，iPhone Safari →「加入主畫面」）。

## 📁 檔案結構
| 檔案 | 說明 |
|---|---|
| `index.html` | 主程式（單檔即整個 App） |
| `_dialogues.json` / `_data/` | 情境對話與資料（來源／重建用） |
| `manifest.json` / `sw.js` | PWA 設定與離線快取 |
| `icon-*.png` / `apple-touch-icon.png` / `icon.svg` | 圖示 |

## 🔒 隱私
完全離線、不連外部伺服器；資料只存裝置 `localStorage`；無帳號、無追蹤、無廣告。適合全家（含未成年）使用。
